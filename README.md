## 简介

**Pica**

使用 Java 开发的微信支付后端服务。

## 快速开始

### 下载最新的 Pica 源码包

```bash
git clone https://github.com/wangjianan1103/pica.git
```

### 启动 Pica

```bash
gradle bootRun
```

## API文档

| 接口地址 | 描述 | 调用方 |
| --- | --- | --- |
| /transaction/do | 业务-下单 | 业务前端 |
| /order/status | 业务-查询订单状态 | 业务前端 |
| /wx/getSdkConfig | 前端初始化jsApi所需配置 | 前端jsApi |
